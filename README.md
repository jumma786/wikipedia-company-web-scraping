# Wikipedia Largest US Companies Web Scraper

## Project Overview
This project uses Python and BeautifulSoup to scrape the list of largest companies in the United States by revenue from Wikipedia.

## Data Extracted
- Rank
- Company Name
- Industry
- Revenue
- Revenue Growth
- Headquarters

## Technologies Used
Python
BeautifulSoup
Requests
Pandas

## How It Works
1. Send HTTP request to Wikipedia page
2. Parse HTML using BeautifulSoup
3. Extract table data
4. Convert to pandas DataFrame
5. Save dataset

## Output Example

Rank | Company | Industry | Revenue | Revenue Growth | Headquarters
--- | --- | --- | --- | --- | ---
1 | Walmart | Retail | $680B | 5.1% | Arkansas
